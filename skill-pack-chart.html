how it works

Fresh Install

Each skill repo is cloned to ~/.claude/packs/<name>/ then its install.sh copies files into ~/.claude/skills/. Restart Claude Code to load all skills.

Re-run = Update

Script detects existing clones in ~/.claude/packs/ and runs git pull on each. Then re-runs install.sh to apply the latest version. Same one-liner every time.

Error Handling

Each of the 5 tools installs independently. If one step fails it prints the error and continues — the rest of the pack still installs. No silent failures.

WP Plugin Step

ZIP is downloaded to ~/Downloads/wp-mcp-ultimate.zip automatically. Then: WP Admin → Plugins → Add New → Upload Plugin. Cloudways users upload via the Cloudways WordPress panel.
