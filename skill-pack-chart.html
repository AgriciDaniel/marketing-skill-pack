Fresh Install

Each skill repo is cloned to ~/.claude/packs/<name>/ then its install.sh is run to place files in ~/.claude/skills/. Restart Claude Code to load all skills.

Re-run = Update

Script detects existing ~/.claude/packs/ clones and runs git pull on each. Then re-runs install.sh to apply the latest version. Same one-liner every time.

Error Handling

Each tool is installed independently. If one step fails it prints the error and continues — the rest of the pack still installs successfully.
